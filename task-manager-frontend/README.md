# Task Manager Frontend

Frontend de una aplicación de gestión de tareas con React, TypeScript y Vite.

[![CI](https://github.com/afpal33/task-manager-frontend/actions/workflows/ci.yml/badge.svg)](https://github.com/afpal33/task-manager-frontend/actions/workflows/ci.yml)

## 🚀 Instalación local

```bash
git clone https://github.com/afpal33/task-manager-frontend.git
cd task-manager-frontend
npm install
```

### Variables de entorno
Las siguientes variables se encuentran en el archivo `.env` en la raíz del proyecto:

```
DATABASE_URL=
JWT_SECRET=
PORT=
```

## 📜 Comandos disponibles

| Comando          | Descripción                              |
|------------------|-------------------------------------------|
| `npm run dev`    | Levanta el entorno de desarrollo           |
| `npm run build`  | Genera el build de producción              |
| `npm test`       | Corre las pruebas automatizadas (pendiente — Sesión 3) |

## 🗄️ Base de datos

PostgreSQL con migraciones y seeds gestionados con Prisma (ver Módulo 2).
